# FHIR App Starter

This is a project starter for your next FHIR App. It is focused on Developer Experience, modern technologies, and best practices.

## Features

- React
- Redux
- Redux-Saga
- Semantic-UI
- Prettier

## Getting Started

```sh
git clone https://github.com/zeevosec/fhir-app-starter
cd fhir-app-starter
yarn install
yarn start
```
